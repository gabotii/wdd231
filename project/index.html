WDD 231 Final Project Site Plan - Gabriel Torrejon

Site Name

Sublima lab - Customizing t-shirts, mugs and much more

This name was chosen because it clearly communicates the main function of the website - offering personalized printing services for various items. It's concise, memorable, and relevant to the project's purpose.

Site Purpose

Sublima lab serves as a comprehensive platform for customers to:

  • Browse and order customized T-shirts, mugs, and other items.
  • Design their own products using an online customization tool.
  • Learn about the different printing techniques available.
  • Access information on pricing and delivery options.
  • Contact the business for bulk orders and special requests.

The site aims to provide an easy and engaging way for customers to create unique, personalized products for themselves or as gifts.

Scenarios

  1. How can I design and order a custom T-shirt with my own artwork?
  2. What are the different printing techniques offered by Sublima lab?
  3. How do I place a bulk order for customized mugs for an event?

Color Schema

The primary color (hsl(212, 89%, 21%)) will be used for main headings and interactive elements like buttons.

The secondary color (hsl(0, 0%, 98%)) serves as the background color for the site, providing a clean and neutral base.

The text color (hsl(0, 0%, 20%)) ensures good readability against the light background.

Typography

Headings: Roboto, sans-serif

Body Text: Open Sans, sans-serif

This combination provides a modern look while ensuring readability across devices.
